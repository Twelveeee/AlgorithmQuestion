Description
最近很火的微信游戏跳一跳游戏规则如下：短跳得1分，跌落游戏结束，长跳得2分，并且长跳可连续累加，第一次2分，第二次4分，依次类推，若长中断则重新计分2分。 现给定0,1,2的组合序列，其中0：跌落 1：短跳成功 2：长跳成功 请你计算玩家当次游戏的合计得分。

Format
INPUT
首先输入一个数字n代表当前游戏状态变化数，接着输入n个由0,1,2组合的数字，0代表结束。
OUTPUT
输出玩家当次游戏得分。
Samples
SAMPLE INPUT
6
1 2 2 1 2 0
SAMPLE OUTPUT
10
Restrictions
Time Limit: 1000 ms

Memory Limit: 65535 KB
