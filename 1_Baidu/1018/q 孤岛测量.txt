描述
小度来到一座孤岛，为了熟悉岛的环境，小度想要测量一下孤岛的周长。假设小度身处在rows*cols大小的二维网格地图grid，如果grid[i][j] == 1则表示陆地，grid[i][j] == 0则表示水域，并且设定岛中间是没有水域的。网格中的格子水平和垂直方向相连（对角线方向不相连）。整个网格被水完全包围，但其中恰好有一个岛屿（或者说，一个或多个表示陆地的格子相连组成的岛屿）。现在需要测量出孤岛，也就是陆地块的周长是多少？

格式
输入格式
行数：int rows 列数：int cols 按行输入每个网格值
输出格式
周长大小：int perimeters
样例
输入样例
4
4
0 1 0 0
1 1 1 0
0 1 0 0
1 1 0 0
输出样例
16
限制
时间限制: 1000 ms

内存限制: 65535 KB

提示
可以手动绘制下二维网格，直观理解，不要输入多个岛屿。
